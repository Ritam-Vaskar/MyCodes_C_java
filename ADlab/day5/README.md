# Day 5: K-Means Clustering and Advanced Clustering Techniques

## Overview
This folder contains implementations of various clustering algorithms applied to different datasets.

## Questions Covered

1. **Q1**: K-Means clustering on Mall Customers Dataset - Customer segmentation analysis
2. **Q2**: DBSCAN on Iris dataset - Noise detection and natural clusters
3. **Q3**: K-Means vs Hierarchical clustering on Wholesale Customers dataset
4. **Q4**: GMM vs K-Means comparison on Iris dataset
5. **Q5**: K-Means on Digits dataset - Handwritten digit grouping
6. **Q6**: Spectral clustering on Iris dataset
7. **Q7**: DBSCAN on Two Moons synthetic dataset
8. **Q8**: Clustering stability analysis on Mall Customers dataset
9. **Q9**: PSO-optimized K-Means on Wine dataset
10. **Q10**: Fuzzy C-Means on Iris dataset

## Installation

```bash
pip install -r requirements.txt
```

## Usage

Run each question individually:
```bash
python q1.py
python q2.py
# ... and so on
```

Or run all questions:
```bash
python run_all.py
```

## Output
Each script generates visualizations and analysis results saved in the respective output folders.
